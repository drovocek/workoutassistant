## Technology stack
1. core:
    - Java 17
    - Spring Boot 3
    - Lombok
    - Gradle
    - JUnit (tests)
2. resource-server:
    - Spring Data JDBC
    - Spring Web MVC
    - PostgreSQL
    - Testcontainers (tests)
3. client-server:
    - Hilla
    - Feign
    - Wiremock (tests)
